Contributing

Contributing nədir?



Contributing termini, Version Control sistemi olan GIT texnologiyasına əsaslanır. Bu proyekt açıq mənbə sayılır, ona görə də öz fikir və qatqılarını GitHub səhifəmizdə bildirə bilərsən!

Contributing şərt və qaydalarını qəbul edirəm.



Zəhmət olmasa qaydaları qəbul etdiyinizi təsdiq edin.
